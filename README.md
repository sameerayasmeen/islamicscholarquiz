# Islamic Scholar Quiz

An interactive Islamic knowledge quiz game designed to test and enhance understanding of Islamic teachings through engaging challenges.

## 🎮 Game Overview
Islamic Scholar Quiz is a web-based game that takes players through four progressive rounds of Islamic knowledge challenges, from basic concepts to advanced visual puzzles.

## 🏆 Game Rounds

### Round 1: Foundational Knowledge (MCQ)
- Multiple choice questions covering Islamic basics
- Topics: Prophets, Quran, basic concepts
- 20 questions

### Round 2: Letter Puzzle
- Fill-in-the-blank puzzles with Islamic terms
- Word completion challenges
- 25 questions

### Round 3: Scramble Master
- Unscramble letters to form Islamic words
- Word rearrangement puzzles
- 30 questions

### Round 4: Visual Quest
- Emoji-based visual representations
- Guess Islamic figures, events, and concepts from sequences
- 15 questions

## 🎯 How to Play
1. Click "Begin Quest" to start
2. Answer questions within the time limit
3. Progress through all 4 rounds automatically
4. View final score and submit to leaderboard

## 🚀 Quick Start
Open `final.html` in any modern web browser to play immediately.

## 🛠️ Technologies
- HTML5, CSS3, JavaScript
- Tailwind CSS for styling
- Supabase for leaderboard (optional)
